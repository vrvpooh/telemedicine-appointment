# Telemedicine & Appointment System 

ระบบที่ให้บริการจัดการนัดหมายระหว่างผู้ป่วยและแพทย์ 

---
## Features
- ระบบจัดการผู้ใช้ 

- ระบบค้นหาเเละข้อมูลเเพทย์ 

- ระบบจัดการตารางเวลา 

- การจองเเละออกตั๋ว 

- ระบบบันทึกผลตรวจ 

- ระบบเเจ้งเตือนเเละประเมินผล
